About

This is a game about two persons (or three if we count you) were you will be asked questions, resolve puzzles. And maybe get different endings.

This game was written in only HTML, CSS, and JavaScript, for a school project.

Warning! This game contains jumpscares, and loud noises, and an overall dark/creepy ambiance!
This game also contains references to panic attacks, and death!

The game also might get data from the computer, but they are NOT shared, and will never be, they are stored temporarily on YOUR computer and/or browser!
